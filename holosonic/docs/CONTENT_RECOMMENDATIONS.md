- Applications list

### Recommended Additions

#### **Technical Details Section**
- **Frequency range**: 20-200 kHz (ultrasonic) - from technical specs
- **Particle size range**: 10 nm to 5 mm - from technical specs
- **Positioning precision**: Sub-10 μm - from technical specs
- **Phased array transducer specifications**: Number of transducers, array configuration
- **Pressure field characteristics**: How pressure nodes are created and controlled
- **Multi-object manipulation**: Technical details on simultaneous control

#### **How It Works (Expanded)**
- **Step-by-step process**:
  1. Phased array transducers generate acoustic waves
  2. Wave interference creates 3D pressure field patterns
  3. Pressure nodes trap particles at specific locations
  4. Dynamic adjustment of phase/amplitude moves particles
  5. Multiple fields can operate independently
- **Visual explanation**: Consider adding diagrams or animations
- **Mathematical foundation**: Brief mention of acoustic field equations (optional, for technical audience)

#### **Comparison Section**
- **vs. Optical tweezers**: 
  - Acoustic works through opaque materials
  - Can handle larger particles
  - Less risk of photodamage
- **vs. Magnetic manipulation**:
  - Works with non-magnetic materials
  - No need for magnetic particles
  - More versatile material compatibility
